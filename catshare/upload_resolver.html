使用说明

关于GitHub访问令牌

您需要创建一个具有repo权限的GitHub个人访问令牌：

  1. 访问GitHub Settings → Developer settings → Personal access tokens
  2. 点击 Generate new token
  3. 选择 repo 权限范围
  4. 生成令牌并在此处使用

上传路径

所有文件将上传到您仓库的指定目录：

/resource/carshare/resolver/

您只需要提供文件名，路径是固定的。

重要注意事项

  • 最大文件大小：25MB（GitHub API限制）
  • 文件名不能包含特殊字符
  • 上传前请确保您的令牌有仓库写入权限
  • 文件将直接提交到指定分支

关于Cloudflare Pages

此工具适用于连接到Cloudflare Pages的GitHub仓库。上传后：

  • Cloudflare Pages会自动检测更改
  • 新版本将在几分钟内完成部署
  • 您可以在Cloudflare Pages仪表板查看部署状态
